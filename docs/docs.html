basic-bot

basic bot is a bot for discord designed to use the fewest amout of libraries. Created so that it is easy to contribute to and is easy for first time discord developers to get into.

https://github.com/spudworthIII/basic-bot/

dependancies

so far there are few dependencies with the ones existing being:

  • nodeJS
  • discord.js
  • FFMPEG
  • Mongoose (by extnsion mongodb)

    • as well as sublibraries contained within those packages

    commands

    So far commands that have been made include

    /ping
    pongs

    /echo < input >
    responds with input

    /kick < user input >
    kicks input user

    /play < name of video >
    searches videos on youtube and plays the audio in voice channel

    /leave
    makes the bot leave the voice channel

    /warn < user input >
    warns the user inputted

    /showwarnings < user input >
    responds with the number of warnings of the user input
